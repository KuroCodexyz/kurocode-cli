# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-06-23

### Added
- Initial release on PyPI as `kurocode-cli`
- One-shot queries with `ask` command
- Interactive chat with `chat` command and slash commands
- Model discovery with `models` command (list, search, info)
- Conversation history with `history` command (list, view, export)
- Configuration management with `config` command
- Profile support for multiple configurations
- Multiple output formats (Rich, Plain, JSON)
- Streaming and non-streaming response modes
- SQLite-backed conversation persistence
- Model caching with disk fallback
- Comprehensive documentation
